Para trabalhar em um projeto com uma equipe seguir os seguintes passos é uma boa recomendação para manter um código bem versionado.

1. Git pull 
2. git checkout -b nomeDaNovaBranch main
3. Adicionar as features na nova branch
4. git add. / git commit
5. git push -> copiar e colar o comando para enviar a branch ao repositório remoto
4. git checkout main
5. git pull
6. git merge nomeDaNovaBranch -> cortar essa etapa no caso de precisar enviar um pull request
7. git push
